What Is the Micro:Bit

| Home | What Is The Micro:Bit | My Project | Core Vaulues & SLA-MS |

A Micro:bit is a compact multipurpose all in one microcontroller, this means that:

  • It can run code that you program for it.
  • It can run one line of code at a time.
  • It has everything you need to run a program (inputs & outputs)

But most of all, the Micro:Bit is a learning tool. It is designed to help students learn how to code.

The Micro:Bit has two main uses these uses are:

Coding Practice

The first, & most common use for the Micro:Bit is learning or practicing coding. The micro bit does a good job of this because that you don't need to worry about attaching other things for your program to work. As well as the fact that the Micro:Bit can use block coding to allow you to get a concept coded without learning how to code JS or Python.

Real World Concepts

While the micro bit is used for education it can also be used for other rudimentary coding projects. Some of these include alarm systems, step counters & Identification Systems.
